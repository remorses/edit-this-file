# Getting started

Edit-this-page Is made up by 2 packages:

-   `babel-plugin-edit-this-page` is a babel plugin that passes your source code to the button component, this is necessary to populate the initial content of the textarea
-   `edit-this-page` is a package that exports the button react component and the modal

The button exported from `edit-this-page` will open up a modal to change the current page source code, on submit a bot will create a PR on the github repo

The pull request will be opened by a Github App, to enable this feature you will need to give the bot access to open pull requests on your repo at [https://github.com/apps/edit-this-page-app](https://github.com/apps/edit-this-page-app)

Here are the steps to enable this feature

> You can try it right now clicking on the "edit this page" button on the top right corner (only visible on desktop)

## 1. Install

Install the dependencies

```sh
yarn add babel-plugin-edit-this-page edit-this-page @chakra-ui/core @emotion/core @emotion/styled
```

Add it to your babel config next to `next.config.js`

```js
// babel.config.js
module.exports = {
    // presets: ['next/babel'], add your personal presets here
    plugins: [
        [
            'edit-this-page',
            {
                editableFiles: 'pages/**',
                githubUrl: 'myusername/myrepo',
                branch: 'master',
            },
        ],
    ],
}
```

## 2. Install the github application

Go to [https://github.com/apps/edit-this-page-app](https://github.com/apps/edit-this-page-app) and install the `Edit This Page` application on your repo

<img src='/edit-this-page/github_app.jpg' width='800px' />

## 3. Add the button

Import and render the button on your react page

The button will open a modal with a textarea to change the page contents, after the users submit a github PR will be opened on Github

```jsx
import EditThisPageButton from 'edit-this-page'

export function App() {
    return (
        <MyApp>
            {/* render the default button */}
            <EditThisPageButton />

            {/* render your own button */}
            <EditThisPageButton unstyled>
                <MyButton>edit this page</MyButton>
            </EditThisPageButton>
        </MyApp>
    )
}
```
